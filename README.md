
## 1. Concept de l'application
# Financial_advisor_chatbot
projet de création d'un chatbot avec langchain et gemini pro API 


## 2. Installation et Exécution (Local)
Pour lancer le projet :
1. Cloner le repo
2. Créer un environnement virtuel : `python -m venv venv`
3. Installer les dépendances : `pip install -r requirements.txt`
4. Ajouter votre clé API dans un fichier `.env` : `GOOGLE_API_KEY=votre_cle`
5. Lancer l'app : `streamlit run app.py`

## 3. Choix Techniques
* **Modèle :** Google Gemini Pro via API
* **Framework :** LangChain (Gestion du prompt et mémoire)
* **Interface :** Streamlit
